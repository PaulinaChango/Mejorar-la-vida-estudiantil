# Mejorar-la-vida-estudiantil
# Nombre del proyecto: StudyEase

## Objetivo
Facilitar la organización de tareas y horarios para estudiantes universitarios.

## Problema que resuelve
Muchos estudiantes olvidan fechas de entrega o tienen mala distribución de su tiempo.

## Público objetivo
Estudiantes de 18 a 25 años de nivel superior.
